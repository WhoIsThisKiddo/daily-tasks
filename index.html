#!/usr/bin/env bash
set -euo pipefail

DIR="/var/log/myapp"
OUT="/tmp/logs-$(date +%Y%m%d).tar.gz"

mkdir -p "$(dirname "$OUT")"
find "$DIR" -type f -mtime -1 -print0 | tar --null -czf "$OUT" --files-from=-
echo "Created archive: $OUT"

Használt szerkezetek: shebang, erre vonatkozó set opciók (errorexit, undefined vars, pipefail), változók, mkdir, find -print0 és tar --null --files-from=- a null-terminated fájlnevek biztonságos átadásához. A script célja: az elmúlt 24 órában módosult naplók összegyűjtése és tömörített archívumba csomagolása.

Használt szerkezetek: változó, Get-ChildItem a fájlok listázásához, pipelining, Sort-Object rendezéshez, Select-Object számított oszloppal a méret MB-ban, és Format-Table megjelenítéshez. A script célja: a megadott mappában található legnagyobb 10 fájl gyors listázása.

Magyarázat: A regex reális mintákat keres: AWS Access Key ID-k (AKIA + 16 nagybetű/szám), valamint 40 karakter hosszúságú Secret-szerű base64-jellegű tokenek. A minta engedi megadott kulcs=érték formát és önálló tokenek találatát is; verbose mód (x) segíti az olvashatóságot. Használható naplóparzolásra és előszűrésre, hogy potenciális hitelesítő adat-szivárgásokat azonosítsunk.
