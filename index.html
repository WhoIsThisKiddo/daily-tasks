#!/bin/bash
# Simple script to list files and directories
echo "Listing current directory contents:"
ls -lah

Ez a Bash script a következőket használja: shebang (#!/bin/bash), echo kiíratás, ls parancs opciókkal. A script célja a jelenlegi könyvtár tartalmának részletes listázása.

# List all processes with memory usage
Get-Process | Sort-Object WS -Descending | Select-Object -First 5

Ez a PowerShell példa a Get-Process parancsot használja a folyamatok lekérdezésére, majd a Sort-Object segítségével memóriahasználat szerint csökkenő sorrendbe rendezi, végül a Select-Object -First 5 a legnagyobb memóriát használó 5 folyamatot jeleníti meg.
